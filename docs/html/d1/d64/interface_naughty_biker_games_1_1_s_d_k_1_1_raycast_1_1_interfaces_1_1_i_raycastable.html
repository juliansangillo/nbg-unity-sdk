GameObject NaughtyBikerGames.SDK.Raycast.Interfaces.IRaycastable.CastRayForTag ( string  tag)

Casts a ray for the given target from the current mouse position. The target is the first Game Object hit by the ray that matches the given tag.

Parameters
tagThe existing tag to target
Returns
The first Game Object hit by the ray that matches the tag, or null if no matching Game Object was hit
