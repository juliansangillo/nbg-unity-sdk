Description

An adapter for Unity's SceneUtility. SceneUtility is a utility class for getting deatils about scenes that are not provided by SceneManager. SceneUtility is a static class and is coupled with Unity's systems. This decouples the SceneUtility from the classes that use it. This way, stubs and mocks of this adapter can be injected for testing purposes.
