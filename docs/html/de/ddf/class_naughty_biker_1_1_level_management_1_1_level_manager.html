◆ Initialize()

void NaughtyBiker.LevelManagement.LevelManager.Initialize ( )

Callback from Zenject.IInitializable that is invoked during Unity's Start phase. When called, it pulls the list of scenes from Unity Build Settings and adds them as levels to track, it sets the first level to level 0, it calculates the labels of the levels, and it sets the active level currently.

This function is called automatically by Zenject. Don't call this directly!
