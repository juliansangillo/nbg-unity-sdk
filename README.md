# Naughty Biker Games Unity SDK
A shared library for Unity 3D that can be integrated with any game project.
## Documentation
Go [here](https://juliansangillo.github.io/nbg-unity-sdk) to view the API docs.
## Source Specifications
**Editor:** Unity 2019.4.12f1 LTS  
**Language:** C#  
**Scripting Backend:** Mono  
**API:** .NET Standard 2.0
