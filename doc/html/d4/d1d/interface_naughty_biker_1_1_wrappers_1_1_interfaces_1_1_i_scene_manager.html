Description

An adapter for Unity's SceneManager. SceneManager is an API that allows you to load new scenes and perform other scene related operations. SceneManager is a static class and is coupled with Unity's systems. This wrapper decouples the SceneManager from the classes that use it. This way, stubs and mocks of this wrapper can be injected for testing purposes.
